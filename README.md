# Real Estate Investment Analyzer

## Summary
An automated data-filtering system designed to scan property zoning logs and land registries to identify under-utilized residential plots optimized for multi-lot subdivision.

## Background
Manual site acquisition and land feasibility checks take days of manual work per property. This system automates the preliminary site selection process, identifying optimal development sites instantly.

## Data and AI techniques
The system uses automated web-scraping to gather municipal spatial records and zoning restrictions. It applies a multi-criteria linear regression model to predict potential Gross Realisation Value (GRV) based on comparable historic plot sales.

## How is it used
The solution runs as a daily automated dashboard utilized by acquisition managers and property wholesalers to screen listings before placing bids.

## Challenges
The system relies heavily on the availability and data freshness of public local council registries. It cannot predict structural geological issues or local council delays.

## What next
Integrating a neural network trained on satellite imagery to automatically filter out plots with massive tree coverage or severe topographic slopes.
